# Автоматическое выделение самостоятельных услуг (IT Purple Avito)

Репозиторий содержит решение бизнес-задачи для вертикали Услуг на Авито (категория «Ремонт и отделка»). Решение помогает бороться с проблемой, когда селлеры описывают в одном объявлении сразу несколько разнородных услуг, из-за чего теряют поисковую выдачу в целевых микрокатегориях.

## Описание решения

Часто исполнители публикуют комплексные объявления (например, "Ремонт под ключ + отдельно электрика и сантехника"). Такое объявление попадает только в основную микрокатегорию. 

Наш сервис (REST API на базе FastAPI) решает эту проблему следующим образом:
1. **Анализ текста**: Принимает исходное объявление и ищет вхождения по словарю ключевых фраз различных микрокатегорий.
2. **Умное разделение (Сплитование)**: С использованием LLM-конвейера сервис извлекает контекст упоминания услуги — является ли она простой составной частью ремонта полного цикла (тогда выделение не нужно), или же исполнитель предлагает ее как самостоятельную работу. 
3. **Генерация черновиков**: Если принято положительное решение о разделении (`shouldSplit = true`), для каждой независимой микрокатегории генерируется персонализированный текст для создания нового узконаправленного черновика объявления.

В пайплайне используется интеграция с **Google AI Studio** (модель семейства Gemini).

## Инструкция по запуску

Проект управляется быстрым пакетным менеджером `uv`.

### 1. Подготовка окружения и установка
Убедитесь, что у вас установлен `uv`. Затем установите все зависимости проекта:
```bash
uv sync
```

### 2. Настройка конфигурации
Создайте файл переменных окружения на основе примера:
```bash
cp .env.example .env
```
Откройте файл `.env` и задайте обязательные настройки:
- **`GEMINI_API_KEY`**: Ваш API-ключ для инициализации клиента Google GenAI. Получить ключ можно в [Google AI Studio](https://aistudio.google.com/apikey).
- **`PROXY_URL`**: В случае сетевых и гео-ограничений (поскольку Google API может быть заблокирован в некоторых регионах) обязательно укажите прокси сервер. Сервис нативно поддерживает SOCKS5 прокси. Например:  
  `PROXY_URL=socks5://127.0.0.1:12334`

### 3. Запуск сервиса
Для старта приложения выполните:
```bash
uv run fastapi dev app/app.py
```
Либо через uvicorn напрямую:
```bash
uv run uvicorn app.app:app --host 0.0.0.0 --port 8080
```
API будет развернуто локально и доступно по адресу: http://localhost:8080/

## Примеры входных данных и работы кода

Примеры взяты из файла-датасета `rnc_dataset.jsonl`.

### Пример 1: Явные самостоятельные услуги (`shouldSplit = true`)

**Входные данные:**
```json
{
  "sourceMcTitle": "Ремонт квартир и домов под ключ",
  "description": "Косметический ремонт под ключ в доме. Отдельно демонтаж сантехники делаем отдельно цементная штукатурка отдельно декоративные конструкции из гкл делаем отдельно потолок в коридоре. частный мастер с опытом 7 лет. стоимость после осмотра. можно сегодня."
}
```
**Результат обработки:**
- Найдены ключевые фразы (категории): `[101, 111, 108, 110, 104]`
- Вердикт о разделении: **ДА** (`shouldSplit = true`)
- Система сгенерирует отдельные тексты объявлений для:
  - Демонтажных работ (111)
  - Штукатурных работ (108)
  - Гипсокартона (110)
  - Натяжных потолков (104)

### Пример 2: Услуги в рамках комплекса ремонта (`shouldSplit = false`)

**Входные данные:**
```json
{
  "sourceMcTitle": "Ремонт квартир и домов под ключ",
  "description": "Ремонт офиса под ключ во вторичке. Гидроизоляция под плитку / финишная отделка под покраску и другие этапы выполняем в составе ремонта. Ищу заказы именно на комплекс, по отдельным видам работ не выезжаю."
}
```
**Результат обработки:**
- Найдены ключевые фразы для разных категорий: `[101, 107, 105]`
- Вердикт о разделении: **НЕТ** (`shouldSplit = false`)
- Поскольку селлер явно указывает, что гидроизоляция и отделка выполняются "в составе ремонта" и по отдельности он не выезжает, дополнительные черновики созданы не будут.

### Пример 3: Единственная моно-услуга (`shouldSplit = false`)

**Входные данные:**
```json
{
  "sourceMcTitle": "Штукатурные работы",
  "description": "Штукатурка по маякам в комната. Делаю нивелирование перепадов, поэтапная оплата. с опытом 8 лет. цена обсуждается. отвечу на вопросы."
}
```
**Результат обработки:**
- Найденные категории: `[108]`
- Вердикт о разделении: **НЕТ** (`shouldSplit = false`)
- Упоминается только профильная работа в рамках одной исходной микрокатегории, сплитование не требуется.
